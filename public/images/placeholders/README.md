# Placeholder images — visual brief

These are temporary images. Replace with commissioned photography when available. The file names are fixed — drop replacement images with the same names and no code change is needed.

When a placeholder file is absent the component renders a warm gradient fallback with the spiral mark, so the layout works at all times without these files present.

---

## hero.webp

- **Used in:** HeroBlock — full-bleed hero band at the top of the homepage
- **Aspect:** roughly 16:7 (very wide crop, landscape). The component clips to fill; exact dimensions at source can be wider.
- **Minimum size:** 1600px wide at 72dpi or 1200px wide at 96dpi. Do not use anything below 1000px wide — it will look soft on retina displays.
- **Subject:** a real person with UK natural curly or Afro-textured hair — 3B, 3C, 4A, 4B, or 4C range. Not a styled product shoot. Documentary register preferred: someone at home, at a wash day, mid-routine. The left third of the image will be obscured by a dark plum gradient overlay and the postcode tool. The right half should be clearly lit with visible hair texture.
- **Treatment:** the left 50% of the image has a dark overlay applied in CSS. Avoid placing the subject's face in the far-left third; it will be partially occluded. Face should sit centre-right on desktop.
- **Sourcing notes:** Unsplash and Pexels. Search terms: "natural hair", "4c hair", "afro hair", "curly hair wash day", "afro textured hair". Favour photos where texture is clearly visible. Avoid stock-photography-looking poses.
